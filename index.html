Gravitational Orbit Optimizing Algorithm

Done for Physics 210 and UBC. I made a gradient descent algorithm that evolutionarily finds stable orbits for an electron around two point mass protons. I was able to characterise three types of orbital paths, Show which types are most likely, using an algorithm I created with 95% accuracy.

Nuclear Fusion Efficiency Simulation

Done for Physics 210 and UBC. I created a simulation that models nuclear fusion using monte carlo methods and the appropriate Maxwell and Gamov Distributions. I concluded that the most energy will be generated the closer we can get to the physical limits of the reactor.
